# Windows Optimizer Pro

Одно приложение для безопасной оптимизации Windows 10/11. Окно с разделами:
автозагрузка, службы, очистка диска, сеть, питание, память, игры, GPU/CPU,
приватность, реестр и **встроенный деблоат** (бывший TurboDebloat — теперь
раздел внутри этого же приложения).

> ⚠️ Интерфейс собирается и проверяется на настоящей Windows через GitHub
> Actions (компиляция, смоук-импорт, сборка `.exe`, тесты безопасности), но
> вживую (нажатие кнопок) ещё не тестировался — первый запуск делайте сами и
> начинайте с **режима проверки / сухого прогона**.

## 🖥️ Скачать (.exe)

**[⬇️ Скачать Windows Optimizer Pro](https://github.com/kirillfomenkov92-crypto/optimikzaion-for-vadik/releases/tag/optimizer-app-latest)**

1. Открыть страницу релиза, под **Assets** нажать `WindowsOptimizerPro.exe`.

3. Windows SmartScreen может предупредить о «неизвестном издателе» — это
   нормально для неподписанного `.exe`: **«Подробнее» → «Выполнить в любом случае»**.

Простая инструкция для не-технического пользователя: [`ДЛЯ_ДРУГА.md`](./ДЛЯ_ДРУГА.md).

## Безопасность

- 🛡️ **Никогда не трогает** Windows Update, Microsoft Defender, Microsoft Store
  и ядро ОС — защита регистронезависима и покрыта тестами.
- 💾 **Бэкап перед изменениями.** Если бэкап реестра не удался — применение
  останавливается. Есть восстановление из бэкапа.
- 🧪 **Сухой прогон по умолчанию** в деблоате: сначала показывает, что *будет*
  сделано, и только потом применяет.
- 📋 Все изменения логируются (модуль, было → стало, результат).

## Разработка

```bash
cd windows_optimizer
pip install -r requirements.txt
python main.py          # на Linux — ограниченный режим (без реальных твиков)
python build.py         # сборка .exe (только на Windows, PyInstaller)
```

Тесты (без зависимости от PyQt6, гоняются в CI):

```bash
python windows_optimizer/tests/test_services_protection.py
python windows_optimizer/tests/test_debloat_safety.py
python windows_optimizer/tests/test_debloat_restore.py
python windows_optimizer/tests/test_tweak_applier.py
python windows_optimizer/tests/test_operation_result.py
```

CI (`.github/workflows/optimizer-ci.yml`) на `windows-latest`: компиляция,
смоук-импорт ядра/деблоата/UI, тесты безопасности, сборка `.exe` и публикация
в Releases при пуше в `main`.
